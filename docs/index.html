자신의 먹BTI 공유가능

자신의 먹BTI를 공유해 그사람과 같이하는 식사자리 선택에 어려움을 줄여보세요.

핵심기술 소개

  • Google Maps, TMap

    주변 식당들을 마커로 표시하고, 자신이 마음에 들었던 식당들을 검색해볼 수 있습니다.

  • Google Places

    식당에 관한 사진들을 통해 어떤 식당인지 유추해볼 수 있습니다.

  • Firebase Real-Time Database

    식당의 먹BTI별 좋아요 순위를 저장하고 관리할 수 있습니다.

팀원 소개

  • 정영광(ELGI)

    ● 금사빠 새우튀김

    ● 먹BTI 테스트 컨텐츠 및 UI 개발

  • 박대호(HMGI)

    ● 혼자 바쁜 민트초코

    ● 검색 기능 및 세부UX 개발

  • 추만석(EMGC)

    ● 익살스런 치아바타

    ● UX 및 데이터저장/관리 개발

  • 강건우()

    ● 일러스트 / UI
